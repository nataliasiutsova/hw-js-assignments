## JavaScript Assignment Operators

| Operator | Example | Same As  |
| -------- | ------- | -------- |
| =        | x=y     | x=y      |
| +=       | x+=y    | x=x+y    |
| -=       | x-=y    | x=x-y    |
| \*=      | x\*=y   | x=x\*y   |
| /=       | x/=y    | x=x/y    |
| %=       | x%=y    | x=x%y    |
| \*\*=    | x\*\*=y | x=x\*\*y |

## Increment/decrement

Increment/decrement can only be applied to variables!

1. Increment ++ increases a variable by 1:

```
let a=2;
a++;
console.log(a); //3
```

2. Decrement -- decreases a variable by 1:

```
let a=2;
a--;
console.log(a); //1
```
